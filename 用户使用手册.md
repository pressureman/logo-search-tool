# Logo 素材助手 · 使用手册

在飞书中直接和机器人对话，用自然语言获取 logo 素材。

---

## 快速开始

直接说出你要的 logo 名称就行，不需要任何指令格式：

```
给我微信的 logo
```

```
我要 3chat logo，PNG 格式
```

```
发我一个蓝色的 Discord 图标
```

```
给我 Figma 的 logo
```

---

## 素材来源

机器人会按以下顺序查找 logo：

1. **内部素材库** — 公司整理的高质量素材，优先使用
2. **SimpleIcons** — 覆盖数千个品牌的开源图标库，自动兜底
3. **Wikimedia Commons** — 维基媒体公共素材库，作为最后备选

内部素材库找不到时，机器人会自动联网搜索，无需手动切换。在线找到的素材会注明来源。

## 指定格式

支持 **SVG、PNG、JPG、WEBP** 四种格式，默认输出 PNG（512px）。

```
给我微信 logo，SVG 格式
给我 telegram 的 webp
给我 Discord 图标 1024px
```

不支持 PDF、GIF 等其他格式。

---

## 改色

支持改色的 logo 可以指定 **单一纯色**（十六进制色号）：

```
给我苹果 logo，改成红色 #FF0000
给我 discord 图标，#6366F1
```

**3Chat 圆形图标**支持圆和图标分开指定颜色：

```
3chat圆形，圆改成蓝色 #1463FF，图标改成白色
```

不支持渐变色、多色、透明度效果。

在线搜索到的 logo 同样支持改色判断——机器人会自动分析素材是否为单色矢量图，能改色的会告知，不能改色的会说明原因。

---

## 多版本选择

部分 logo 有多个版本（如 3Chat 有默认、圆形、横版三个版本），机器人会列出选项，你可以：

- 回复序号：`1` / `第二个` / `3`
- 回复版本名：`圆形版` / `横版`
- 一次全要：`都给我` / `全部`（会打包成 ZIP 文件发送）

---

## 固定多色 Logo

部分 logo 使用品牌固定多色（如 Google、飞书、微软），不支持改色，机器人会告知并询问是否需要原版。

## 对话示例

**获取内部素材库 logo：**
> 用户：给我微信 logo  
> 机器人：发送微信绿色图标（PNG 512px）

**获取在线 logo：**
> 用户：给我 Figma 的 logo  
> 机器人：在 SimpleIcons 找到了，支持改色，默认发 PNG，需要指定颜色或尺寸吗？此素材来自 SimpleIcons，非公司内部素材库  
> 用户：默认就行  
> 机器人：发送 Figma logo（PNG 512px）

**改色：**
> 用户：我要 discord 图标，改成黑色 #000000  
> 机器人：发送黑色 Discord 图标

**多版本选择：**
> 用户：给我 3chat logo  
> 机器人：有3个版本，你要哪个？  
> 用户：都给我  
> 机器人：发送 logos.zip（含全部3个版本）

**不支持的改色：**
> 用户：给我微软 logo，改成蓝色  
> 机器人：微软 logo 是固定多色版本，不支持改色，需要原版吗？  
> 用户：要  
> 机器人：发送原版微软 logo

## 常见问题

**Q：发了消息没反应？**  
A：稍等几秒，机器人在处理。如果超过 30 秒没有回复，可以重新发一次。

**Q：说了 logo 名字机器人没找到？**  
A：机器人会自动联网搜索，如果仍然找不到，可以试试提供品牌的英文名再问一次。

**Q：在线找到的 logo 和内部素材库有什么区别？**  
A：内部素材库的素材经过人工整理，质量更稳定；在线素材为自动抓取，机器人会注明来源，质量因品牌而异。

**Q：想取消当前操作？**  
A：直接说"不要了"或"算了"，机器人会取消并等待新的请求。
